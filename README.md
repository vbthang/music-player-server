# Base ExpressJS

Base ExpressJS is a basic source code, using Express FrameWork

## Requirements

- node ~ v18.17.0
- npm ~ v9.6.7

## Usage

1. Clone project
2. Create `.env` file, copy content from [.env.example](./.env.example) to `.env` file and config in `.env`

3. Install package & setup

```bash
npm install
```

4. Runs the app

```bash
npm run dev
```
